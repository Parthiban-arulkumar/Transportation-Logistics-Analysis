# Transportation & Logistics Performance Analysis Dashboard 🚚📊

## Project Overview
This Power BI project analyzes transportation and logistics operations to monitor shipment performance, on-time deliveries, delayed deliveries, route efficiency, and driver analysis.

The dashboard helps stakeholders identify operational bottlenecks and improve logistics performance through data-driven insights.

---

## Business Problem
Logistics companies often face challenges such as:
- Delivery delays
- Inefficient route planning
- Shipment tracking issues
- Poor visibility into operational KPIs

This dashboard provides actionable insights to improve transportation efficiency and customer satisfaction.

---

## Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Excel
- Data Visualization
- Business Intelligence

---

## Key KPIs
- Total Shipments
- On-Time Deliveries
- Delayed Deliveries
- On-Time Delivery %
- Delay Delivery %
- Average Trip Distance

---

## Dashboard Pages

### 1️⃣ Overview Analysis
- Shipment KPIs
- Delivery trends
- Driver performance analysis
- Vehicle delay analysis
- Route analysis

### 2️⃣ Time Analysis
- Monthly shipment trends
- Order trends over time
- Quarterly delivery analysis
- On-time vs delayed deliveries

### 3️⃣ Route Analysis
- Longest delivery locations
- Least delivery locations
- Shipment category analysis
- Route distribution insights

---

## Key Insights
- 6K shipments analyzed
- 70.45% on-time deliveries
- 29.55% delayed deliveries
- Identified high-performing and low-performing routes
- Analyzed shipment distribution patterns across locations

## Business Impact
This dashboard helps businesses:
- Improve delivery performance
- Reduce transportation delays
- Optimize logistics operations
- Monitor KPIs effectively
- Support business decision-making
